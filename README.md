# 🌸 Dorami Cursor Set: The "Logic over Habit" Collection

[English] | [日本語] | [Tiếng Việt]

A professional, UX-driven cursor set inspired by Dorami. Designed with visual anchors and emotional feedback to replace boring system defaults.

ドラミちゃんにインスパイアされた、UX重視のプロフェッショナルなカーソルセットです。視覚的なアンカーと感情的なフィードバックを考慮し、退屈なシステム既定のカーソルを置き換えるために設計されました。

Bộ con trỏ chuột chuyên nghiệp hướng tới trải nghiệm người dùng (UX) lấy cảm hứng từ Dorami. Được thiết kế với các điểm neo thị giác và phản hồi cảm xúc nhằm thay thế các con trỏ mặc định nhàm chán.

### 🛠 Installation / インストール / Cài đặt

1.  **[EN]** Right-click `install.inf` -> **Install**. Go to Mouse Properties -> Pointers -> Select **Dorami** scheme.
2.  **[JP]** `install.inf` を右クリックして「インストール」を選択。マウスの設定から「ドラミ」スキームを選択。
3.  **[VI]** Chuột phải vào `install.inf` -> chọn **Install**. Vào Mouse Properties -> Pointers -> Chọn Scheme **Dorami**.
